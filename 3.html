Задачи на состояние state и события в React

На стейты

Задание 1. Дан следующий стейт: this.state = {name: 'Иван', age: 25}; Сделайте так, чтобы метод render вывел на экран следующее ('Иван' и 25 должны взяться из стейта):
имя: Иван, возраст: 25

На события

Задание 2. Сделайте кнопочку, по нажатию на которую сработает метод showMessage и на экран алертом выведется сообщение с текстом 'hello'.

Задание 3. Дан следующий стейт: this.state = {name: 'Иван', age: 25}; Сделайте кнопочку, по нажатию на которую сработает метод showName и на экран алертом выведется имя пользователя из стейта

На setState

Задание 4. Дан следующий стейт: this.state = {name: 'Иван', age: 25}; Изначально метод render выводит на экран следующее:
имя: Иван, возраст: 25
Сделайте кнопочку, по нажатию на которую 'Иван' поменяется на 'Коля', а 25 поменяется на 30.
